# 🔍 TermuSearch - Simple Search Engine

A simple frontend-based search engine built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🔎 Search functionality
- ⚡ Fast local search
- 🎨 Clean UI
- 📱 Beginner friendly

## 📁 Project Structure
index.html
style.css
script.js
data.js
README.md
LICENSE
.gitignore
## 🧠 How it Works

- `data.js` contains searchable data
- `script.js` filters data based on user input
- Results are displayed dynamically on screen

## ▶️ How to Run

1. Download or clone this repo
2. Open `index.html` in browser
